# Slack Sales Dashboard

Sales performance tracking and lead management dashboard.

## Install

```bash
npm install --legacy-peer-deps
```

## Run

```bash
npm run dev  # http://localhost:5173
```

## Build

```bash
npm run build
```

## Deploy

Vercel auto-deploys from `main` branch.
